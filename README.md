### Final Project
* This project is a number guessing game that utilizes the `random` module
* The program asks the user to enter a lower and upper bound
* Based on these bounds, the user is able to guess a number in between
* The program logs the attempts and displays the number of guesses
* If the user decides they want to play again, they are welcome to do so
---
### Problem Statement
* Create a simple and fun game utilizing multiple _Python_ functions
* The game doesn't necessarily solve a problem, but enables many useful and crucial programming skills
* Users are able to customize / choose their experience
---
### Logic / Algorithmic / Code Explanation
* First, there is a welcome message displayed to explain the rules of the guessing game
* The project uses the `random` module to randomly generate a number between the lower and upper bounds generated by the user
* Functions are utilized to ensure structure and cleanliness
* Input validation ensures that all entered integers are actually integers
* Input validation also raises a value error if the lower bound is greater than or equal to the upper bound
* A while loop is utilized to rerun the game while the user hasn't guessed the number yet
* If conditions take note of the guess in terms of it is is higher or lower than the random number
* Attempt count is stored in a counter variable after every loop interation
* After the user guesses the number, the attempts are displayed and they are prompted to be able to play again
* If they select yes, the game restarts. If they select no, the program is terminated
